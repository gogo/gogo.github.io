Overview

gogoprotobuf is a fork of golang/protobuf with several extra features which fall into different categories:
  • Code Generation
  • Libraries
  • Tools

Code Generation

gogoprotobuf uses the Extensions feature of Protocol Buffers to achieve:
  • fast marshalling and unmarshalling.
  • more canonical Go structures.
  • goprotobuf compatibility
  • less typing by optionally generating extra helper code.
  • peace of mind by optionally generating test and benchmark code.
  • other serialization formats

Libraries

  • fieldpath creates and compiles fieldpaths for selective unmarshalling.
  • parser parses a proto file with a fork-exec of protoc (does not work on windows).
  • io provides length delimited readers and writers.

Tools

  • pbpath
  • letmegrpc
